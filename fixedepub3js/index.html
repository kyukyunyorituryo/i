Fixed Layout Epub3.jsの使い方

2026/5/27

FixedEpub3JSの使い方

簡単に漫画のEPUBを作れる 電子出版初心者向けの取説。

かんたんEPUB作成

漫画や写真集などの画像が主な固定レイアウトのEPUBを簡単に作れるサイト。

  • Chrome推奨
  • 画像999枚以下
  • 200MB程度まで確認済み 対応OS:
  • Windows
  • Android
  • Mac
  • iPadOS

使い方

画像形式は JPG と PNG に対応。

  • 見開きページは分割が必要
  • 白ページで位置調整可能 見開き確認は EPUB ビュワーで確認してください。

本の綴じ方向

通常の漫画は右綴じ。

  • 左タップで次ページ
  • 左スワイプで次ページ

パネルビュー

Kindle端末でダブルタップ時の拡大移動方向を指定できます。

通常漫画

四コマ

本の情報入力

  • タイトル
  • 著者名1
  • 著者名2 著者名2は省略可能。

表紙画像

表紙画像を選択すると：

  • 横幅
  • 高さ が自動設定されます。 対応形式:
  • JPG
  • PNG

本文ファイル読み込み

複数画像を一括読み込み可能。 注意点:

  • HTML5仕様上、読み込み順はランダム
  • 「画像整列」でファイル名順に並び替え推奨 Android / iPadOS では制限があります。

目次編集

EPUBの目次はページジャンプ機能です。 例:

  • 表紙
  • 目次
  • 第一話
  • 奥付 など。

EPUBビュワー

おすすめ:

  • Readium
  • BiB/i Kindle確認は Kindle Previewer 推奨。

JSライブラリについて

  • 著者

  • 目次 などをオブジェクト化し、ライブラリとして汎用化したいと考えています。
